# my-first-api

我的第一个 RESTful API，用 Node.js + Express 构建。

## 接口列表

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/users | 获取用户列表 |
| GET | /api/users/:id | 获取单个用户 |
| POST | /api/users | 创建新用户 |
| PUT | /api/users/:id | 更新用户 |
| DELETE | /api/users/:id | 删除用户 |

## 本地运行

```bash
npm install
npm start
```

访问：http://localhost:3100
